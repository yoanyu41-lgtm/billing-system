================================================================================
            CITYTECH BILLING SYSTEM - ROUTES, MENU & SYSTEM BUILD
================================================================================

System: CityTech Computer Shop - Installment Management System
Framework: Laravel 10+ (PHP)
Frontend: Blade Templates + Tailwind CSS (CDN)
Authentication: Laravel Breeze + Sanctum (API)
Database: MySQL / MariaDB
Languages: English (en) + Khmer (km)

================================================================================
                          SYSTEM BUILD / TECH STACK
================================================================================

BACKEND:
--------
- Framework        : Laravel 10+ (PHP 8.1+)
- ORM              : Eloquent
- Authentication   : Laravel Breeze (web) + Laravel Sanctum (API tokens)
- Database         : MySQL / MariaDB
- Queue System     : Laravel Queue (notifications)
- Task Scheduling  : Laravel Scheduler (cron jobs)
- File Storage     : Laravel Storage (public disk)

FRONTEND:
---------
- Template Engine  : Blade
- CSS Framework    : Tailwind CSS (via CDN)
- Icons            : Font Awesome 6.4.0
- Fonts            : Khmer OS Siemreap (Khmer + English)
- Theme            : Light/Dark mode support (forced light)

INTEGRATIONS:
-------------
- Telegram Bot API : Customer notifications & payment reminders
- PDF Generation   : Invoice & contract printing
- Database Backup  : Automated backup/restore system

KEY FEATURES:
-------------
- Multi-language (English/Khmer) with locale switching
- Role-based access control (admin/staff/user)
- Global search (customers + products)
- Real-time notifications
- Dashboard analytics (revenue & collection charts)
- Soft deletes for customers

MIDDLEWARE:
-----------
- auth          : Requires authentication
- admin         : Admin-only access (AdminMiddleware)
- guest         : Non-authenticated users only
- auth:sanctum  : API token authentication
- SetLocale     : Language localization

ARTISAN COMMANDS:
-----------------
- BackupDatabase.php          : Database backup automation
- SendDuePaymentReminders.php : Payment reminder notifications
