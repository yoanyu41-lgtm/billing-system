# មុខងារការកំណត់ក្រុមហ៊ុន (Company Settings)

## ទិដ្ឋភាពទូទៅ
មុខងារការកំណត់ក្រុមហ៊ុន អនុញ្ញាតឱ្យអ្នកគ្រប់គ្រងព័ត៌មានក្រុមហ៊ុនរបស់អ្នកពីផ្ទាំងគ្រប់គ្រង។ ព័ត៌មាននេះនឹងត្រូវប្រើប្រាស់ដោយស្វ័យប្រវត្តិនៅពេលបោះពុម្ពកិច្ចសន្យាបង់រំលស់ ដែលធ្វើឱ្យវាងាយស្រួលក្នុងការថែរក្សាកិច្ចសន្យាឱ្យទាន់សម័យដោយមិនចាំបាច់ផ្លាស់ប្តូរកូដ។

## មុខងារសំខាន់ៗ

### ការគ្រប់គ្រងព័ត៌មានក្រុមហ៊ុន
- **ឈ្មោះក្រុមហ៊ុន** (អង់គ្លេស និង ខ្មែរ)
- **អាសយដ្ឋាន** (អង់គ្លេស និង ខ្មែរ)
- **ព័ត៌មានទំនាក់ទំនង** (ទូរស័ព្ទ និង អ៊ីម៉ែល)
- **លេខអាជ្ញាប័ណ្ណពាណិជ្ជកម្ម** (ស្រេចចិត្ត)
- **ឡូហ្គោក្រុមហ៊ុន** (ស្រេចចិត្ត - JPG, JPEG, PNG រហូតដល់ 2MB)

### ការរួមបញ្ចូលកិច្ចសន្យាស្វ័យប្រវត្តិ
នៅពេលអ្នកបោះពុម្ពកិច្ចសន្យាបង់រំលស់ វានឹងទាញយកព័ត៌មានក្រុមហ៊ុនចុងក្រោយពីមូលដ្ឋានទិន្នន័យដោយស្វ័យប្រវត្តិ:
- ឈ្មោះក្រុមហ៊ុនលេចឡើងនៅក្នុងបឋមកថា និង បាតកថា
- អាសយដ្ឋានបង្ហាញនៅក្នុងផ្នែកអ្នកលក់
- ទូរស័ព្ទ និង អ៊ីម៉ែល បង្ហាញនៅក្នុងផ្នែកទំនាក់ទំនង

## របៀបប្រើប្រាស់

### ចូលប្រើការកំណត់ក្រុមហ៊ុន

1. ចូលប្រព័ន្ធជា **អ្នកគ្រប់គ្រង (Admin)**
2. ចុចទៅ **ការកំណត់ (Settings)** ពីរបារចំហៀង
3. ចុចប៊ូតុង **🏢 ការកំណត់ក្រុមហ៊ុន** នៅជ្រុងខាងស្តាំខាងលើ

### ធ្វើបច្ចុប្បន្នភាពព័ត៌មានក្រុមហ៊ុន

1. បំពេញទម្រង់ជាមួយព័ត៌មានលម្អិតក្រុមហ៊ុនរបស់អ្នក
2. បញ្ចូលឡូហ្គោ (ស្រេចចិត្ត) - ទទួលយក JPG, JPEG, PNG រហូតដល់ 2MB
3. ចុច **💾 រក្សាទុកការផ្លាស់ប្តូរ**
4. សារជោគជ័យនឹងលេចឡើងបញ្ជាក់ការធ្វើបច្ចុប្បន្នភាព

### មើលការផ្លាស់ប្តូរក្នុងកិច្ចសន្យា

1. ទៅកាន់គម្រោងបង់រំលស់ណាមួយ
2. ចុច **បោះពុម្ពកិច្ចសន្យា**
3. កិច្ចសន្យានឹងបង្ហាញព័ត៌មានក្រុមហ៊ុនដែលអ្នកបានធ្វើបច្ចុប្បន្នភាព

## ព័ត៌មានបច្ចេកទេស

### ការផ្ទុកក្នុងមូលដ្ឋានទិន្នន័យ
ការកំណត់ក្រុមហ៊ុនត្រូវបានរក្សាទុកក្នុងតារាង `settings` ដោយប្រើគូ key-value:

| Key | ការពិពណ៌នា |
|-----|-------------|
| `company_name` | ឈ្មោះក្រុមហ៊ុនជាភាសាអង់គ្លេស |
| `company_name_km` | ឈ្មោះក្រុមហ៊ុនជាភាសាខ្មែរ |
| `company_address` | អាសយដ្ឋានជាភាសាអង់គ្លេស |
| `company_address_km` | អាសយដ្ឋានជាភាសាខ្មែរ |
| `company_phone` | លេខទូរស័ព្ទទំនាក់ទំនង |
| `company_email` | អាស័យដ្ឋានអ៊ីម៉ែលទំនាក់ទំនង |
| `company_business_license` | លេខអាជ្ញាប័ណ្ណពាណិជ្ជកម្ម (ស្រេចចិត្ត) |
| `company_logo` | ផ្លូវទៅកាន់ឯកសារឡូហ្គោដែលបានបញ្ចូល (ស្រេចចិត្ត) |

### Routes (ផ្លូវ)
- **មើលការកំណត់:** `GET /admin/settings/company`
- **ធ្វើបច្ចុប្បន្នភាពការកំណត់:** `POST /admin/settings/company`

## ការគ្រប់គ្រងឡូហ្គោ

### ការបញ្ចូលឡូហ្គោ
1. ចុច **បញ្ចូលឡូហ្គោថ្មី** នៅក្នុងទំព័រការកំណត់ក្រុមហ៊ុន
2. ជ្រើសរើសឯកសាររូបភាព (JPG, JPEG, PNG)
3. ទំហំឯកសារអតិបរមា: 2MB
4. ចុច **រក្សាទុកការផ្លាស់ប្តូរ**

### ការរក្សាទុកឡូហ្គោ
- ឡូហ្គោត្រូវបានរក្សាទុកនៅក្នុង `storage/app/public/company/`
- ឡូហ្គោចាស់នឹងត្រូវលុបដោយស្វ័យប្រវត្តិនៅពេលបញ្ចូលឡូហ្គោថ្មី
- ផ្លូវឡូហ្គោត្រូវបានរក្សាទុកក្នុងមូលដ្ឋានទិន្នន័យសម្រាប់ការទាញយកងាយស្រួល

## អត្ថប្រយោជន៍

✅ **មិនចាំបាច់ផ្លាស់ប្តូរកូដ** - ធ្វើបច្ចុប្បន្នភាពព័ត៌មានក្រុមហ៊ុនតាមរយៈផ្ទាំងគ្រប់គ្រង  
✅ **គាំទ្រពីរភាសា** - អង់គ្លេស និង ខ្មែរ សម្រាប់ទាំងឈ្មោះក្រុមហ៊ុន និង អាសយដ្ឋាន  
✅ **ការធ្វើបច្ចុប្បន្នភាពកិច្ចសន្យាស្វ័យប្រវត្តិ** - កិច្ចសន្យាទាំងអស់នាពេលអនាគតប្រើព័ត៌មានចុងក្រោយបំផុត  
✅ **គាំទ្រឡូហ្គោ** - បន្ថែមឡូហ្គោក្រុមហ៊ុនរបស់អ្នកសម្រាប់ការបង្កើតម៉ាកយីហោវិជ្ជាជីវៈ  
✅ **ងាយស្រួលប្រើប្រាស់** - ចំណុចប្រទាក់ទម្រង់សាមញ្ញជាមួយការផ្ទៀងផ្ទាត់  

## ការកែលម្អនាពេលអនាគត

ការកែលម្អដែលអាចធ្វើបានសម្រាប់កំណែនាពេលអនាគត:
- ប្រវត្តិរូបក្រុមហ៊ុនច្រើន (សម្រាប់អាជីវកម្មដែលមានសាខាច្រើន)
- ជម្រើសទីតាំងឡូហ្គោនៅក្នុងកិច្ចសន្យា
- វាលទំនាក់ទំនងបន្ថែម (ទូរសារ, គេហទំព័រ, ប្រព័ន្ធផ្សព្វផ្សាយសង្គម)
- ជម្រើសប្ដូរតាមបំណងគំរូ
- ការប្ដូរតាមបំណងបឋមកថា/បាតកថាកិច្ចសន្យា
